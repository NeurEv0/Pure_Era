🗒️ SPADE 快速笔记

SPADE 快速笔记是一个功能强大的悬浮式笔记工具，让您在浏览数据库时随时记录想法和发现。 所有笔记都保存在本地，确保您的隐私和数据安全。

自动保存

笔记内容实时保存到本地，无需手动操作，跨页面访问

自由拖拽

可以自由移动笔记窗口位置，适应您的工作习惯

折叠展开

支持折叠和展开，节省屏幕空间，提高工作效率

导出下载

支持下载笔记为文本文件，方便备份和分享

快捷键

支持 Ctrl+Shift+N 快捷键快速打开/关闭笔记

隐私保护

所有数据仅保存在本地，不会上传到任何服务器

🔧 API 使用示例

如果您是开发者，可以通过以下 API 来控制笔记功能：

// 打开笔记 window.quickNotes.show(); // 关闭笔记 window.quickNotes.hide(); // 切换显示/隐藏 window.quickNotes.toggle(); // 获取笔记内容 const content = window.quickNotes.getContent(); // 设置笔记内容 window.quickNotes.setContent('新的笔记内容'); // 追加内容 window.quickNotes.appendContent('\n追加的内容'); // 检查是否显示 const isVisible = window.quickNotes.isShown();

📋 使用说明

  1. 点击导航栏中的 图标打开快速笔记
  2. 使用快捷键Ctrl + Shift + N快速开启/关闭
  3. 在文本区域输入您的笔记内容，系统会自动保存
  4. 拖拽标题栏来移动窗口到合适的位置
  5. 使用-按钮折叠窗口，节省空间
  6. 使用↓按钮下载笔记为文本文件
  7. 使用🗑按钮清空所有笔记内容
  8. 使用×按钮关闭笔记窗口

🌟 特色功能

  • 跨页面同步：在任何 SPADE 页面上的笔记都会自动同步
  • 实时字符计数：实时显示笔记字符数量
  • 边界检测：拖拽时自动限制在屏幕范围内
  • 状态记忆：记住窗口位置和折叠状态
  • 响应式设计：在移动设备上同样好用
