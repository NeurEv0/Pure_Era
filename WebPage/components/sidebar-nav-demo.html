侧边栏导航组件概述

这是一个独立、可复用的侧边栏导航组件，专为 SPADE 系统设计。组件采用现代Web标准构建，具有简约的设计风格和完整的响应式支持。

🎮 快速体验

📦 文件结构

components/ ├── sidebar-nav.css # 样式文件 (~8KB) ├── sidebar-nav.js # 功能脚本 (~12KB) ├── sidebar-nav-demo.html # 演示页面 └── README.md # 详细说明文档

功能特性

智能导航

自动处理导航项的活动状态，支持URL哈希值同步，提供平滑滚动体验。

响应式设计

完美适配桌面、平板、手机等各种设备，提供一致的用户体验。

键盘友好

支持键盘导航，符合Web可访问性标准，提升使用体验。

易于定制

使用CSS变量进行主题配置，支持自定义颜色、尺寸和动画效果。

高性能

事件节流优化、智能缓存、最小重排重绘，确保流畅的交互体验。

即插即用

自动初始化，无需手动配置，引入文件即可使用。

API 参考

🎯 核心方法

// 访问组件实例 const sidebar = window.sidebarNav; // 设置活动导航项 sidebar.setActiveNavItem(element); // 切换侧边栏显示状态 sidebar.toggleSidebar(); // 打开/关闭侧边栏 sidebar.openSidebar(); sidebar.closeSidebar(); // 平滑滚动到指定部分 sidebar.smoothScrollToSection('#section-id'); // 获取当前活动项 const activeItem = sidebar.getCurrentActiveItem(); // 启用基于滚动的导航 sidebar.enableScrollBasedNavigation();

响应式测试

调整浏览器窗口大小来测试组件在不同设备上的表现：

桌面端 (>992px)

完整侧边栏显示，悬停展开文字标签。

平板端 (768-992px)

侧边栏适度缩小，保持核心功能。

手机端 (<768px)

侧边栏隐藏，显示切换按钮，点击展开。
