顶部导航组件演示

这是一个专门用于处理顶部导航栏移动端菜单切换的独立组件。组件提供简约的设计风格，与网站主题色保持一致。

🎮 快速体验

📦 组件特点

  • 自动适配：根据屏幕尺寸自动切换显示模式
  • 简约设计：与网站主题色保持一致的视觉风格
  • 触摸友好：44px的触摸目标，符合移动端规范
  • 无障碍支持：完整的键盘导航和ARIA属性
  • 高性能：使用CSS3变换和事件节流优化

功能特性

响应式设计

自动检测屏幕尺寸，在移动端显示汉堡菜单按钮，桌面端保持水平布局。

平滑动画

菜单展开/收起具有流畅的过渡动画，菜单项依次出现效果。

活动状态

自动根据当前页面设置活动菜单项，支持手动设置。

键盘导航

支持方向键、Home、End键导航，ESC键关闭菜单。

覆盖层

菜单展开时显示半透明覆盖层，点击外部区域自动关闭。

滚动锁定

移动端菜单打开时禁止页面滚动，防止误操作。

📱 测试说明

调整浏览器窗口大小来测试不同断点下的表现：

  • >768px: 桌面端，水平导航栏
  • ≤768px: 移动端，汉堡菜单

API 参考

🎯 核心方法

// 访问组件实例 const headerNav = window.headerNav; // 切换菜单显示状态 headerNav.toggleMenu(); // 打开/关闭菜单 headerNav.openMenu(); headerNav.closeMenu(); // 设置活动菜单项 const menuItem = document.querySelector('.nav-menu a[href="search.html"]'); headerNav.setActiveMenuItem(menuItem); // 获取状态信息 const state = headerNav.getMenuState(); console.log('菜单状态:', state.isOpen); console.log('活动项:', state.activeItem);

使用示例

📝 HTML 结构

<!-- 引入组件样式 --> <link rel="stylesheet" href="components/header-nav.css"> <!-- 顶部导航栏结构 --> <header> <div class="logo-container"> <a href="#" class="logo">SPADE</a> </div> <ul class="nav-menu"> <li><a href="home.html">首页</a></li> <li><a href="search.html">搜索</a></li> <li><a href="tools.html">工具</a></li> </ul> </header> <!-- 引入组件脚本 --> <script src="components/header-nav.js"></script>

🎨 CSS 变量配置

:root { --sidebar-bg: #2b1055; /* 导航栏背景色 */ --sidebar-hover: #7597de; /* 悬停/活动色 */ --header-height: 80px; /* 顶栏高度 */ }

⚡ JavaScript 使用

// 组件自动初始化后可通过 window.headerNav 访问 document.addEventListener('DOMContentLoaded', function() { // 设置自定义事件监听 window.addEventListener('resize', () => { console.log('窗口大小变化'); }); // 手动控制菜单 const customButton = document.getElementById('custom-toggle'); customButton.addEventListener('click', () => { window.headerNav.toggleMenu(); }); });
